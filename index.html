Featured Project

Offline Maize Yield Prediction System

An intelligent offline-first decision support system designed to help smallholder farmers predict maize yield using Machine Learning and IoT. The platform integrates ESP32 sensors, SQLite database storage, and a Random Forest model to generate real-time yield predictions without an internet connection.

IoT Integration

Interactive Dashboard

Provides farmers with prediction results, historical records, visual insights and an easy-to-use interface for decision making.
